### Árvore Binária

**Descrição do Problema:**
Implemente uma estrutura de árvore binária e escreva funções para percorrer a árvore em pré-ordem, em ordem e em pós-ordem.

**Exemplo:**
```
Input:            1
                /   \
               2     3
              / \
             4   5

Pre-ordem: [1, 2, 4, 5, 3]
Em ordem:  [4, 2, 5, 1, 3]
Pós-ordem: [4, 5, 2, 3, 1]

```
